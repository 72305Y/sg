Git

  • INIT - Создает новый репозиторий Git
  • CLONE - Копирует существующий репозиторий Git
  • STATUS - Отображает состояние рабочего каталога и раздела проиндексированных файлов
  • ADD - Первая команда в цепочке операций, предписывающей Git «сохранить» снимок текущего состояния проекта в истории коммитов
  • PUSH - Позволяет передать изменения из локального репозитория (набора файлов из папки .git) в удаленный
